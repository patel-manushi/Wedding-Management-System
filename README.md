# Wedding-Management-System
<p align="justify">
The Wedding Management System is a web-based application that simplifies and organizes wedding planning on a single platform. It allows users to securely register and log in, explore and book wedding services based on their requirements, access professional consultancy support, submit inquiries through a contact form, and view a gallery of completed wedding events, while the admin panel efficiently manages services, pricing, and bookings for a smooth planning experience.
</p>
